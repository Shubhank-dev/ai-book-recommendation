# Screenshots

Run the app locally with:

```bash
streamlit run app.py
```

Then capture and save screenshots of each page into this folder using the following filenames so they match the references in the main project README:

- `home.png` — Home page (hero section + metrics + featured books)
- `content_based.png` — Content-Based Recommendation page
- `personalized.png` — Personalized Recommendation page
- `trending.png` — Trending Books page
